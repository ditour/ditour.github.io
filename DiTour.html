DiTour App
The DiTour enterprise app for iPad is an award winning and disruptive digital signage app. It was deployed at SNS in November, 2012 as a cost saving, intuitive and reliable replacement compared to the previous third party solution. A single iPad replaces three pieces of expensive hardware from the previous system. Today, this app has gained popularity of its own merit with multiple site deployment around the lab.
Description
  • Displays a default slideshow in a continuous loop on a connected monitor
  • A track may be any sequence of images and videos
  • Modern and intuitive touch display for selecting a track to display (tour mode)
  • Automatically reverts back to the default slideshow after a delay
Version 4.1 Features
  • Runs on iPhone and iPad
  • Media is loaded from a remote server and stored on the device
  • Supports png, jpeg and gif image, PDF, web page, mp4 video and COLLADA dae slide formats
  • Multiple presentations can be stored on the device
  • Smart directory traversal for zero configuration remote media managment from http or https sites
  • Updates only the files it needs
  • Updates files in the background whether playing or if the app is in the background
  • Automatically dim the display with the following user selectable options
    • Never: never automatically dim the screen
    • On Battery Power: (default mode) monitors the battery state and dims the screen if on battery power and no activity for 5 minutes
    • Inactive - Off Hours Only: Same as "On Battery Power" and will also dim the screen during off hours (user specified) and no activity for 30 minutes
    • Inactive: Same as "On Battery Power" plus will also dim the screen after no activity for 30 minutes regardless of time of day
  • Improved rendering of web slides
  • Web slides can be forced to resize by calling resizeDiTourSlide() in JavaScript from the webpage
  • System logging allows for periodic logging of status (DiTour heartbeat, current track, battery state and level and more) to a Graylog server
  • Source code in Swift 2.2
  • Various bug fixes and user interface improvements
Future

  • iBeacon support for automated tours
  • Account for daylight savings time change when checking for updates (check the actual file information)
  • Automatic periodic file updates with smart status check
